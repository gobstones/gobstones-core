Enumeration Direction

This enum represent the valid Gobstones Directions. It's accompanied by a namespace with the same name, that provides additional functionality, such as asking for the first or the last direction, or iterate over the elements of this enum.

Note that directions are sorted in the following order, from first to last.

  • Direction.North
  • Direction.East
  • Direction.South
  • Direction.West

Always prefer using the enum over the string values it represents, even as object keys.
